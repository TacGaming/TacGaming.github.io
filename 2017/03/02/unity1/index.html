Unity3D 基础课堂

2017-03-02
Unity

架构

每个项目为一个Project，每个Project包含很多Scene，Scene只管来说就是游戏中的场景、关卡之类的，当然也包括开始画面等等。

Scene中包括很多GameObject,Unity中GameObject是一个引擎写好的类，用以表示游戏世界中的所有游戏对象（包括物体、人物、摄像机、灯光等等）不同类型的游戏对象是通过GameObject下不同的Component组件来区分的。

几何体预设

在大纲视图点击右键可以看到3D Object的字样，其中可以选择在场景中新建很多预设的几何体。

调整游戏对象的属性

选中物体后，可以通过左上角的5个工具来修改属性，也可以直接修改右侧属性面板中的Transform组件数值来改变游戏对象的位置、大小、旋转等

让物体感受到物理引擎

Unity中的游戏对象如果想要使用物理引擎，就一定要具有Rigidbody刚体组件，可以在选中物体后在右侧点击Add Component中找到RigidBody来添加

添加代码

在资源面板邮件点击，找到Create -> C#，创建一个以类名命名的脚本。并将脚本挂载到物体上。

脚本中Start函数为该物体生成时执行的函数。

Update函数为游戏进行时每帧会执行的函数

我们在Update函数中添加下面的代码：

public float speed = 1;

void Start() {

}

void Update() {
  if(Input.GetKey(KeyCode.W)) {
    transform.Translate(new Vector3(0,0,speed * Time.deltaTime));
  }
}

使用Prefab预设

新建一个物体，并把物体拖动到下面的资源面板，这样它就成为了一个Prefab预设

我们对代码进行一些修改：

public float speed = 1;

public GameObject BulletPrefab;
void Start() {

}

void Update() {
  if(Input.GetKey(KeyCode.W)) {
    transform.Translate(new Vector3(0,0,speed * Time.deltaTime));
  }

  if(Input.GetKeyDown(KeyCode.Space)) {
    Inisiate(BulletPrefab,transform.position,transform.rotation);
  }
}
